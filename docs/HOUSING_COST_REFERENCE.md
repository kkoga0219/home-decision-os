> Home Decision OS の計算ロジック根拠・改善指針としてまとめた技術ドキュメント。
> Zillow の機能体系を参考に、日本市場に適応した形で整理。

## 1. Zillow の機能体系（参考モデル）

Zillow は米国最大の住宅プラットフォームで、以下の計算ツールを提供している。
Home Decision OS が目指すべき機能の参考として整理する。

### 1.1 Zestimate（物件価格推定）

ニューラルネットワークベースの推定モデル。1.68億件以上の物件データを保有。

入力データ: 面積、所在地、間取り、MLS掲載価格、周辺取引実績、
税務評価額、過去の売買履歴、季節変動トレンド

精度: 掲載中物件で中央誤差率1.9%、非掲載物件で約7.0%

→ Home Decision OS への示唆:
MLIT取引価格データ + SUUMO掲載データを教師データとし、
LightGBM/XGBoostでの㎡単価予測モデルを構築する方向は正しい（TODO #8）。
Zillowのように「取引実績」「税務評価額」「季節変動」を特徴量に含めるべき。

### 1.2 Rent Zestimate（賃料推定）

1.15億件の賃料データをベースに、物件属性・周辺賃料相場・取引価格から
月額賃料を推定。

入力データ: 面積、間取り、所在地、周辺賃料、最終取引価格

→ Home Decision OS への示唆:
現在の3層推定（SUUMO実データ→ブレンド→利回り逆算）に加え、
MLIT取引データの取引件数を「流動性指標」として組み込むと精度向上が見込める。

### 1.3 Mortgage Calculator（月次支出内訳）

Zillowの住宅ローン計算機は以下の項目を月額で内訳表示する:

- Principal + Interest（元金+利息）
- Property Tax（固定資産税を月割）
- Homeowners Insurance（火災・地震保険を月割）
- HOA Fees（管理費・修繕積立金に相当）
- PMI（Private Mortgage Insurance、頭金20%未満の場合。日本にはない）

→ Home Decision OS への示唆:
現在のCFシミュレーターは年次表示。月次内訳のビジュアル表示（円グラフ等）を
追加すると直感的に理解しやすくなる。

### 1.4 Home Sale Calculator（売却手取り計算）

売却時の手取り額を計算。売却価格から以下を控除:

- 仲介手数料（米国: 売却額の3〜6%）
- Title Insurance（権利保険: 0.5%程度）
- 譲渡所得税
- ローン残債
- その他諸費用

→ Home Decision OS への示唆:
現在の出口シナリオは「売却諸費用4%」で一括計算しているが、
日本の仲介手数料（3%+6万+税）と譲渡所得税（3000万控除考慮）を
分けて計算すべき。後述のセクション4で詳細。

### 1.5 Affordability Calculator（購入可能額算出）

年収・既存ローン・頭金から、購入可能な物件価格を逆算。
返済負担率（DTI）の上限を自動設定。

→ Home Decision OS への示唆:
「この年収ならいくらの物件が買えるか」の逆算機能は未実装。
日本のフラット35基準（返済負担率30〜35%以下）で実装可能。

## 2. 購入時の諸費用（Initial Costs）

中古マンション購入時の諸費用は、物件価格の6〜9%が目安。

### 2.1 仲介手数料

計算式（宅建業法上限）:

| 物件価格帯 | 計算式 |
|---|---|
| 800万円超 | 物件価格 × 3% + 6万円 + 消費税(10%) |
| 400万円超〜800万円以下 | 物件価格 × 3% + 6万円 + 消費税(10%) |
| 200万円超〜400万円以下 | 物件価格 × 4% + 2万円 + 消費税(10%) |
| 200万円以下 | 物件価格 × 5% + 消費税(10%) |
| 800万円以下（低廉空家特例、2024年7月〜）| 上限30万円 + 消費税 |

例: 5,000万円の物件 → (5,000万 × 3% + 6万) × 1.1 = 171.6万円

現在の実装値: `price * 0.03 + 60,000) * 1.10` → 正しい

### 2.2 登録免許税（登記費用）

マンション購入時に必要な登記:

| 登記の種類 | 本則税率 | 軽減税率（〜2027年3月） | 課税標準 |
|---|---|---|---|
| 土地 所有権移転 | 2.0% | 1.5% | 固定資産税評価額 |
| 建物 所有権移転（中古） | 2.0% | 0.3% | 固定資産税評価額 |
| 抵当権設定 | 0.4% | 0.1% | 債権額（借入額） |

軽減措置の条件:
- 床面積50㎡以上
- 自己居住用
- 取得後1年以内の登記
- 新耐震基準適合（1982年以降築 or 耐震証明）

司法書士報酬: 5〜10万円

例: 5,000万円の物件（評価額: 市場価格の70%として3,500万円、借入4,500万円）
- 土地移転: 3,500万 × 50%（土地按分） × 1.5% = 26.25万円
- 建物移転: 3,500万 × 50%（建物按分） × 0.3% = 5.25万円
- 抵当権: 4,500万 × 0.1% = 4.5万円
- 司法書士: 約8万円
- 合計: 約44万円（物件価格の約0.9%）

現在の実装値: `price * 0.008` = 40万円 → 概ね妥当だがやや低め。
改善案: 土地・建物・抵当権を分けて計算し、軽減措置の適用可否を判定。

### 2.3 不動産取得税

基本: 固定資産税評価額 × 4%（住宅用は軽減で3%、〜2027年3月）

中古マンションの軽減措置:

建物: 固定資産税評価額から築年数に応じた控除額を差し引き
| 新築年月 | 控除額 |
|---|---|
| 1997年4月1日〜 | 1,200万円 |
| 1989年4月1日〜1997年3月 | 1,000万円 |
| 1985年7月1日〜1989年3月 | 450万円 |

土地: 以下のいずれか大きい方を税額から控除
- 4.5万円
- (土地1㎡あたり評価額 × 1/2) × (住戸床面積 × 2、上限200㎡) × 3%

条件: 床面積50〜240㎡、自己居住用、新耐震基準適合

例: 5,000万円の中古マンション（2010年築、評価額3,500万円）
- 建物: (3,500万 × 50% - 1,200万) × 3% = max(0, 7.5万) = 7.5万円
- 土地: 評価額ベースで計算、多くの場合は軽減措置で0円に近い
- 合計: 10〜20万円程度

現在の実装値: `assessed * 0.03 / 2` → 概ね妥当だが、築年数による控除額の違いを反映すべき。

### 2.4 住宅ローン関連費用

| 費用項目 | メガバンク | ネット銀行 |
|---|---|---|
| 事務手数料 | 3〜5万円 | 借入額 × 2.2% |
| 保証料 | 借入額 × 約2%（35年の場合） | 0円 |
| 印紙代 | 1〜2万円 | 電子契約なら0円 |
| 団信（団体信用生命保険） | 金利に含む | 金利に含む |

例: 4,500万円借入（35年）
- メガバンク: 事務手数料3万 + 保証料約93万 + 印紙2万 = 約98万円
- ネット銀行: 事務手数料99万 + 保証料0 + 印紙0 = 約99万円

現在の実装値: `loan_amount * 0.02` → メガバンクの保証料に近い。
改善案: メガバンク型/ネット銀行型のプリセットを用意。

### 2.5 その他初期費用

| 項目 | 目安 |
|---|---|
| 印紙税（売買契約書） | 1〜3万円（物件価格による） |
| 固定資産税精算金 | 年額の日割り |
| 修繕積立基金（新築のみ） | 20〜50万円 |
| 火災・地震保険（5年一括） | 8〜15万円 |
| 引越し費用 | 10〜30万円 |

## 3. 保有期間中のランニングコスト

### 3.1 固定資産税・都市計画税

計算の基本構造:

固定資産税 = 課税標準額 × 1.4%（標準税率）
都市計画税 = 課税標準額 × 0.3%（上限税率）

マンションの場合の特徴:
- 建物と土地を別々に計算する
- 土地は「マンション全体の敷地 ÷ 戸数」が持分
- 固定資産税評価額は3年ごとに見直し（直近は2024年度）

建物の評価額:
- 新築時: 再建築価格 × 経年減点補正率
- RC造の場合、築20年で新築時の約50〜60%程度まで逓減
- 下限は新築時評価額の約20%（完全にゼロにはならない）
- 市場価格に対する評価額の割合は概ね50〜70%

土地の軽減措置（住宅用地の特例）:

| 区分 | 固定資産税 | 都市計画税 |
|---|---|---|
| 小規模住宅用地（200㎡以下/戸） | 課税標準を1/6に | 課税標準を1/3に |
| 一般住宅用地（200㎡超/戸） | 課税標準を1/3に | 課税標準を2/3に |

マンションは1室=1戸としてカウント。多くの場合、持分面積は200㎡以下なので
小規模住宅用地の特例が適用される。

新築マンションの建物軽減:
- 新築から5年間（認定長期優良住宅は7年間）、建物の税額が1/2に
- 適用期限: 2026年3月31日までに取得した物件

実際の税額感覚（中古マンションの場合）:

| 物件価格 | 築年数 | 固定資産税+都市計画税（年額目安） |
|---|---|---|
| 3,000万円 | 築10年 | 8〜12万円 |
| 3,000万円 | 築20年 | 6〜10万円 |
| 5,000万円 | 築10年 | 13〜18万円 |
| 5,000万円 | 築20年 | 10〜15万円 |
| 7,000万円 | 築10年 | 18〜25万円 |

現在の実装の評価:
- 建物: `price * 0.30 * age_factor` → age_factorの逓減率0.025/年、下限0.30
  → 築20年で30%×50% = 15%。実際の評価額/市場価格比は30〜50%程度なので
  やや低めだが、マンション按分を考慮すると妥当な範囲
- 土地: `price * 0.10` に1/6特例 → 実質1.67%
  → マンションの土地持分の小ささを反映しており妥当
- 改善案: 「固定資産税通知書の金額を直接入力」するオプションを設けるのが最も正確

### 3.2 管理費・修繕積立金

マンション固有の費用。市場の目安:

| 項目 | 全国平均 | 都市部（関西） |
|---|---|---|
| 管理費 | 10,000〜15,000円/月 | 12,000〜18,000円/月 |
| 修繕積立金 | 8,000〜15,000円/月 | 8,000〜12,000円/月 |

注意点:
- 修繕積立金は段階増額方式が多く、築年数とともに上がる
- 大規模修繕時に一時金が発生することもある（50〜100万円）
- 現在の実装は固定値だが、年率1〜2%の増額を見込むべき

### 3.3 火災・地震保険

| 項目 | 目安（マンション） |
|---|---|
| 火災保険（5年一括） | 3〜8万円 |
| 地震保険（5年一括） | 5〜10万円 |

マンションは構造がRC/SRCのため、木造戸建てより大幅に安い。
現在の実装値: 15,000円/年 → 妥当

## 4. 売却時の費用と税金

### 4.1 売却諸費用

| 項目 | 金額 |
|---|---|
| 仲介手数料 | 売却額 × 3% + 6万円 + 消費税 |
| 印紙税 | 1〜3万円 |
| 抵当権抹消登記 | 1〜2万円 |
| ローン繰上返済手数料 | 0〜5万円 |

合計目安: 売却額の3.5〜4%

現在の実装値: `sale_price * 0.04` → 妥当

### 4.2 譲渡所得税

売却益に対する税金。計算式:

譲渡所得 = 売却価格 - 取得費 - 譲渡費用

| 所有期間 | 税率（所得税+住民税+復興税） |
|---|---|
| 5年以下（短期譲渡所得） | 39.63% |
| 5年超（長期譲渡所得） | 20.315% |

※ 所有期間は「売却した年の1月1日時点」で判定。
  例: 2020年4月購入 → 2025年12月売却でも、2025年1月1日時点で4年8ヶ月 → 短期

居住用3,000万円特別控除:
- 自己居住用物件なら、譲渡所得から3,000万円を控除できる
- 所有期間の長短を問わず適用可能
- 住宅ローン控除との併用は不可（どちらか選択）

例: 5,000万で購入、10年後に4,300万で売却（取得費4,800万円とする）
- 譲渡所得 = 4,300万 - 4,800万 - 170万（諸費用）= マイナス → 課税なし

実態: 日本のマンションは多くの場合、経年で価格が下がるため
譲渡所得税が発生しないケースが多い。ただし都心の一等地や
タワーマンションなど値上がりした場合は注意。

現在の実装: 譲渡所得税は未考慮。
改善案: 出口シナリオに「3,000万控除後の課税判定」を追加。
値上がりした場合のみ税額をマイナス表示。

### 4.3 取得費の計算

取得費 = 購入価格 + 購入諸費用 - 建物の減価償却累計額

自己居住用でも売却時には減価償却を計算する必要がある:
- RC造マンション: 耐用年数70年（非事業用は事業用の1.5倍）
- 償却率: 0.015（定額法）
- 建物取得費 × 0.9 × 0.015 × 経過年数 = 減価償却累計

## 5. 住宅ローン控除（減税制度）

### 5.1 制度概要（2025〜2026年）

年末のローン残高の0.7%を所得税（+住民税の一部）から控除。

| 住宅タイプ | 借入限度額（2025年） | 控除期間 |
|---|---|---|
| 長期優良住宅・低炭素住宅（新築） | 4,500万円 | 13年 |
| ZEH水準省エネ住宅（新築） | 3,500万円 | 13年 |
| 省エネ基準適合住宅（新築） | 3,000万円 | 13年 |
| その他新築 | 0円（2024年以降は対象外） | - |
| 長期優良等（中古） | 3,000万円 | 10年 |
| その他中古 | 2,000万円 | 10年 |

子育て世帯・若者夫婦世帯（19歳未満の子がいる、又は夫婦いずれかが40歳未満）
は限度額が上乗せされる。

### 5.2 2026年度の主な変更

- 新築の省エネ基準適合: 限度額2,000万円に縮小（2025年は3,000万円）
- 中古の省エネ基準適合以上: 控除期間が10年→13年に拡充
- 中古の長期優良等: 限度額3,500万円に拡大（2025年は3,000万円）
- 子育て世帯向け上乗せ措置の継続

### 5.3 適用条件

- 自己居住用であること（投資用は対象外）
- 床面積50㎡以上（合計所得1,000万円以下なら40㎡以上）
- 合計所得金額2,000万円以下
- 返済期間10年以上
- 取得後6ヶ月以内に入居

### 5.4 控除額の実際

例: 中古マンション（省エネ基準非適合）、借入残高3,500万円
- 限度額2,000万円 × 0.7% = 14万円/年（実際の残高ではなく限度額が上限）
- 10年間の控除合計: 最大140万円（実際は残高逓減で少し少ない）

現在の実装:
- 控除率0.7%、中古10年/新築13年、年間上限21万円
- 改善案: 省エネ基準の有無による限度額の違いを反映。
  年間上限は「限度額×0.7%」で計算すべき（一般中古なら14万円）。

## 6. 投資用不動産のCF計算

### 6.1 減価償却の正しい計算

RC造マンション（投資用）の耐用年数と償却率:

法定耐用年数が全て経過している場合:
- 耐用年数 = 法定耐用年数 × 20% = 47 × 0.2 = 9年
- 償却率: 0.112

法定耐用年数の一部が経過している場合:
- 耐用年数 = (47 - 経過年数) + 経過年数 × 20%
- 例: 築15年 → (47-15) + 15×0.2 = 35年、償却率0.029
- 例: 築30年 → (47-30) + 30×0.2 = 23年、償却率0.044

建物価格の按分:
- 消費税額から逆算する方法が最も正確（消費税は建物にのみ課税）
- 建物価格 = 消費税額 ÷ 10%
- 消費税額不明の場合: 固定資産税評価額の比率で按分
- 概算: マンションの場合、物件価格の50〜70%が建物（立地による）

現在の実装値: `price * 0.60` → 妥当な概算

### 6.2 不動産所得と損益通算

不動産所得 = 総収入金額 - 必要経費

必要経費に含まれるもの:
- 減価償却費
- ローン利息（元金返済は含まない）
- 管理費・修繕積立金
- 固定資産税・都市計画税
- 火災保険料
- 賃貸管理委託費（家賃の5%程度）
- 不動産所得にかかる税理士費用

損益通算:
- 不動産所得が赤字の場合、給与所得等と通算して所得税を減額可能
- ただし「土地取得のためのローン利息」は損益通算の対象外
- 建物部分のローン利息のみ損益通算可能

現在の実装の評価:
- 損益通算のロジックは前回修正で正しく実装済み
- ローン利息を経費に含めていない（保守的）→ 将来的に追加を検討
- 改善案: ローン利息の建物按分を経費に含めると、より正確な節税効果を算出可能

|---|---|---|
| 子育てグリーン住宅支援事業（2025） | 省エネ住宅の新築・リフォーム | 160万円（GX志向型） |
| みらいエコ住宅2026事業 | 省エネ住宅の新築・リフォーム | 125万円 |
| 給湯省エネ2025事業 | 高効率給湯器導入 | 最大16万円 |
| 先進的窓リノベ2025事業 | 窓の断熱改修 | 最大200万円 |

注意: 中古マンション購入で直接使える補助金は限定的。
リフォーム・リノベーションと組み合わせることで活用可能。

### 7.2 自治体独自の補助金

尼崎市・西宮市等、各自治体が独自の住宅取得支援制度を設けている場合がある。
- 子育て世帯の住宅取得助成
- 中古住宅購入+リフォーム助成
- 定住促進補助金

→ Home Decision OS への示唆:
自治体の補助金情報は変更が頻繁なため、自動取得は困難。
「補助金チェックリスト」としてユーザーに確認を促すUIが現実的。
